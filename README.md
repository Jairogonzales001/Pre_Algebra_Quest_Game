# Algebra Quest: The Inequality Challenge

An educational RPG game for learning two-step linear inequalities, aligned with Florida Standards MA.8.AR.2.2 and MA.K12.MTR.2.1.

## Game Files

- **index.html** - Main HTML structure with all game screens and UI elements
- **styles.css** - All game styling, animations, and visual effects
- **game.js** - Complete game logic and mechanics

## How to Play

1. Open `index.html` in a web browser
2. Choose your dungeon difficulty:
   - **Whispering Forest (Easy)** - Positive coefficients only
   - **Crystal Caves (Medium)** - Includes negative numbers
   - **Shadow Tower (Hard)** - Master level with frequent inequality reversals
3. Solve two-step inequalities to defeat enemies
4. Level up and face progressively harder problems

## Key Features

- **Progressive Difficulty**: Problems scale with your level
- **Three Dungeons**: Each with unique difficulty and enemy types
- **Level-Scaled Enemies**: Enemies deal more damage as you level up
- **Shop System**: Buy health potions, power gems, and hint scrolls
- **Auto-Disappearing Messages**: Battle log cleans itself after 15 seconds
- **Mobile-Friendly**: Fully responsive design works on phones and tablets
- **PWA Support**: Install on your device's home screen for offline play

## Answer Format

Type solutions like: `x>5` or `x<=-3`
- Use `<`, `>`, `<=`, or `>=` for inequality symbols
- No spaces required
